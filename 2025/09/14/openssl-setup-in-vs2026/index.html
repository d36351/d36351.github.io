openssl_setup_in_vs2026

2025/9/14 openssl

這是關於openssl配置的教學

這個教學參考自這個教學影片 (本文撰寫時間為2025-09-14)

環境:

我們需要先下載openssl

選擇 最新版本的EXE(這裡是3.5.2)
注意不要選擇light

Visual studio 配置:

新建個c++空白檔
在資源檔新增一個name.cpp 貼上以下程式

#include <openssl/ssl.h>
#include <openssl/err.h>
#include <iostream>

int main() {
    SSL_library_init();
    SSL_load_error_strings();
    std::cout << "OpenSSL initialized successfully!" << std::endl;
    return 0;
}

看到這個

點右鍵，滑到最下面，選擇”屬性”
在C/C++子目錄之中的一般，看到”其他include 目錄”
看到最右邊的/，選擇編輯
新增Openssl子目錄之下的include 加入其中

回到屬性，選擇連接器(linker)下的一般
找到”其他程式庫目錄”仿照之前的操作
新增Openssl子目錄之下的lib/VC/x64 看到旗下有四個資料夾
將其全部新增

選擇連接器下的”輸入”
選擇”其他相依性”
輸入

libssl.lib  
libcrypto.lib

這樣就完成了
