推薦使用之網站

2025/9/2 github

這篇是有關在學習可能有所助益的資源以及個人的經驗

以下提供在學習之時可能會有幫助的網站

1.學習方向相關

對於擬定自身學習計畫及排定時辰有所助益的網站或文章

程式設計:

學習程式設計 此網站提供多種語言之程式設計課程

資訊科學:

學習資訊科學 此網站提供大量不同領域的文章捷徑，適合剛踏入一個領域時作為參考 包含: 物理引擎(phyical engine) 3D渲染器(3D render) 擴增實境(AR) 資料庫(database) 遊戲(game) 機器人(bot) 程式語言(programming language) shell 網路伺服器(web sever) 網路瀏覽器(web browser) 搜尋引擎(search engine) 數據庫(data base) git 神經網路(neural network) docker

等

數學:

2.學習輔助相關

對於在學習時提供更加有效的捷徑的網站或軟體

3.潛規則相關

提問相關:

對於踏入某個領域的新人常無法理解的黑話、術語，以及提問時可以求助的方向 提問方式

此項目提供對於同領域專家提問的技巧與方法 摘要: 對於答者保持尊重 重於描述問題本身，避免使用情緒性字眼 完整描述問題發生的經過 在提問前確定網路上其他資源無法解決你的問題

對於某個github上的特定專案，可以藉由issue功能 本網站的儲存庫issue鏈接 一般而言其練接為 專案網址/issues

注意: 對於本站提供之學習方法僅供參考，個人學習情況有所不同，謹慎鑑別。
