LaTeX 常用數學

2025/9/6

本篇介紹常用數學表達式

表達式:

使用$ ("shift"+"4")
$在這裡輸入表達式$
$$雙重金錢號表示此行獨立且置中$$

右上標:

使用^ ("shift"+"6")
如: $ x^2 $

右下標:

使用_ ("shift"+"-")
如: $ x_2$

若標之中之字元數>=2，則需加入{ }
如:
$ x^{20} $

$ x_{20} $

否則:
$ x^20 $
$ x_20 $
將變成:

根號:

使用\sqrt\["num"]
其中num 填入num 次方根，若為平方則不需填
如:
$ \sqrt x $
$ \sqrt[3] x $

注意’’ 與x之間需加入空格
同理，若根號內之字元數>=2，則需加入{}

分號

使用 \frac{分子}{分母}
如:
$\frac{2}{3}$

如果需在其外加上括號，需要加上\left 及\right

如:
$\left( \frac{2}{3}$ \right)$
) $
否則:
$( \frac{2}{3}$ )$
) $
